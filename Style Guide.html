/Abstract 10 chapters · ≈ 22 min read

A working reference for everything that holds this field manual together. Tokens, the type system, drafting conventions for diagrams, the component catalogue, and a set of hard-working dos & don'ts that came out of building the artefact itself — not pieties.

The rule is consistency under restraint. Black ink on white paper does almost all the work; vermilion is rationed; the line work is hairline; the type stays out of the way. Read the don'ts first if you only have one minute.
